**RoPieee**

The latest RoPieee (2019/09/22) can be downloaded from [here](http://image.ropieee.org/20190922-ropieee-ose_rpi234-stable.bin)

XZ compressed version of the image (about 1/3th of the original size) [here](http://image.ropieee.org/20190922-ropieee-ose_rpi234-stable.bin.xz)

**RoPieeeXL**

The latest RoPieeeXL image (2019/09/10) can be downloaded from [here](http://image.ropieee.org/20190922-ropieeexl-ose_rpi234-stable.bin)

XZ compressed version of the image (about 1/3th of the original size) [here](http://image.ropieee.org/20190922-ropieeexl-ose_rpi234-stable.bin.xz)

**Flashing**

If you use [Etcher](https://etcher.io) for flashing an SD card you can use this XZ compressed image *directly* without first decompressing it.

*Limitations*

- RoPieee requires a Raspberry Pi 2, 3 or 4.
- With wireless (Pi 3/4) we only support DHCP.
